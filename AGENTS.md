# AGENTS.md — Project Preferences

## Output Format Preference

- **Always use HTML presentations** for educational/teaching content
- Dark theme, keyboard navigation (arrow keys, spacebar), progress bar
- 14+ slides per presentation — more content is welcome if needed
- Include: title slide, overview, concept slides, comparisons, quick reference, closing slide
- Do NOT use Reveal.js or external dependencies — self-contained single HTML files
- Do NOT default to markdown-only deliverables — HTML presentation is the primary format
