Overview

The Landscape

AI development tools fall into three categories:

🖥️ IDE Extensions

Plugins that live inside your code editor. See your code, edit files, run commands.

Roo Code Cline

⌨️ CLI Tools

Terminal-based AI assistants. No GUI needed. Great for servers and remote work.

Claude Code OpenCode Gemini CLI

🔧 Frameworks

Libraries to build your own custom AI agents and workflows.

Open Manus

IDE Extension

Roo Code

VS Code extension that acts as an AI pair programmer with autonomous capabilities.

Capabilities

  • Reads entire codebase for context
  • Generates and edits code directly
  • Runs terminal commands
  • Debugs errors from stack traces
  • Multiple API provider support
  • Autonomous plan → execute → verify

Best For

  • Day-to-day coding assistance
  • Refactoring existing code
  • Writing tests
  • Understanding unfamiliar code
Setup
Install from VS Code Marketplace → Add API key (OpenRouter recommended) → Select model → Start coding

IDE Extension

Cline

Autonomous AI coding agent for VS Code with browser automation and MCP support.

Capabilities

  • Everything Roo Code does, plus:
  • Browser automation — click, fill, scrape
  • Full MCP support
  • Create entire project structures
  • Web search for documentation
  • Self-correction from errors

Best For

  • Building projects from scratch
  • Web scraping tasks
  • Browser interaction workflows
  • Independent autonomous work
Setup
Install from VS Code Marketplace → Configure API → Enable browser & MCP in settings → Give it a task

CLI Tool

Claude Code

Anthropic's official CLI tool, designed specifically for codebase interaction.

Capabilities

  • Deep codebase understanding
  • Multi-file edits across projects
  • Runs tests and fixes failures
  • Git operations (commit, branch, diff)
  • Natural language task execution
  • Strong reasoning abilities

Best For

  • Large codebase refactoring
  • Complex multi-file changes
  • Code review and analysis
  • When you need reasoning over speed
# Install npm install -g @anthropic-ai/claude-code # Start interactive session claude

CLI Tools

OpenCode, Gemini CLI, OpenClaw

⌨️ OpenCode

Open-source CLI AI coding assistant. Supports multiple API providers. Fully customizable.

Best: Terminal workflows, server/remote dev, open-source preference

Free (open source) + API costs

⌨️ Gemini CLI

Google's CLI for Gemini models. Multimodal (text + images). GCP integration. Generous free tier.

Best: Google ecosystem, multimodal AI, GCP tasks

Generous free tier available

⌨️ OpenClaw

Open-source CLI AI tool. Lightweight, extensible. Good for custom workflows and scripting.

Best: Lightweight CLI, custom workflows, open-source stack

Free (open source) + API costs

Framework

Open Manus

Open-source Python framework for building custom AI agents.

Capabilities

  • Build agents with specific skills
  • Define workflows and task pipelines
  • Integrate with any API provider
  • Custom tool creation
  • Multi-agent coordination

Best For

  • Building your own AI agents
  • Custom automation workflows
  • When off-the-shelf tools don't fit
  • Learning how agents work
git clone https://github.com/mannaandpoem/OpenManus cd OpenManus pip install -r requirements.txt # Configure API keys and run

Advice

Tips for Juniors

1

Start with free models

Don't pay until you've exhausted free tiers

2

Be specific with prompts

"Fix the login bug" is better than "fix bugs"

3

Review every change

AI makes mistakes. Always read what it writes.

4

Use the right tool

CLI for server work, IDE extensions for coding

5

Learn the basics first

Don't let AI write code you don't understand

6

Set up OpenRouter first

One key works with all tools. Start with free models.
